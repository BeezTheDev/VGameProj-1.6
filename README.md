> **This project is not affiliated with, endorsed by, sponsored by, or associated with Riot Games or VALORANT.**

## How to Mod

1. Install Epic Games Launcher.
2. Download and install Unreal Engine 4.23.1.
3. Clone or download this UProject.
4. Open the project in Unreal Engine 4.23.1.
5. Make your changes, and create your own content.
6. Build and test.

Have fun modding!
